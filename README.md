# Syntaric Landing Page

A professional static landing page for Syntaric - Healthcare Interoperability Consulting.

## Features

- Modern, responsive design
- Clean and professional layout
- Mobile-friendly
- Services showcase
- Expertise highlights
- Contact information
- Sticky navigation

## Structure

- `index.html` - Main landing page with embedded CSS
- `etc/` - Contains company logos and documentation
  - `icon_only.png` - Company icon used in navigation
  - `syntaric_logo_2.png` - Full company logo
  - `syntaric_12.2025.pdf` - Company information document

## Viewing the Page

Simply open `index.html` in any modern web browser.

## Deployment

This is a static HTML page that can be deployed to any web hosting service:

- GitHub Pages
- Netlify
- Vercel
- Traditional web hosting (Apache, Nginx, etc.)

No build process or dependencies required.

## Customization

All styling is embedded in the `<style>` section of `index.html`. Key design variables are defined in the `:root` CSS selector for easy customization of colors and branding.
